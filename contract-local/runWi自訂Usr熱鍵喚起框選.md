# I. 主旨目的

定義使用者於任意前景活動（含無邊框視窗化遊戲）中，以全域熱鍵喚起選區遮罩並完成框選的單一行為。

# II. 參考準備

* 前置：[sysScreenTrans系統] 已依 [setWi自訂Usr啟動結束常駐] 常駐於系統匣。
* 使用前提：查詢對象若為遊戲，須以無邊框視窗化（borderless windowed）執行；獨占全螢幕（exclusive fullscreen）下遮罩無法覆蓋。

# III. 內容程序

1. 使用者按下 `Alt+L`（左右 Alt 皆可）。
2. 全螢幕出現半透明變暗遮罩（<300ms），游標變十字，遮罩上顯示一行操作提示。
3. 使用者按住滑鼠左鍵拖曳，出現高對比橡皮筋選框；放開左鍵即完成選取。
4. 任一時點按 `ESC`：取消選取、遮罩消失、回到喚起前狀態。

**可驗證結果**：熱鍵按下後遮罩如期出現；放開左鍵後系統取得與螢幕實際像素對位正確之選區影像（多螢幕與 DPI 縮放環境亦然）；`ESC` 取消後無殘影視窗。
