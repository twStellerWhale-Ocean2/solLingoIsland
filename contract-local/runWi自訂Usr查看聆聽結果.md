# I. 主旨目的

定義使用者於查詢完成後，在浮動結果視窗查看三項結果、聆聽原文發音並關閉視窗的單一行為。

# II. 參考準備

* 前置：[runWi自訂Sys辨識翻譯選區] 已回傳合格之 [datIntf自訂查詢結果格式] 結果。
* 朗讀引擎依 [techItem語音合成]（Windows 內建語音）。

# III. 內容程序

1. 結果視窗出現於選區旁（不遮擋原選區），分三區顯示：英文原文、KK 音標、繁體中文翻譯。
2. 使用者點擊播放按鈕：以系統英文語音非同步朗讀原文；重複點擊時先停止前次播放再重播。
3. 使用者按 `ESC` 或點擊視窗外部：視窗關閉，回到遊戲。

**可驗證結果**：三區內容與查詢結果一致；播放呼叫可被測試攔截驗證（不實際發聲）；`ESC` 關閉後無殘影視窗。

# IV. 備註紀錄

* 2026/7/3：依 solScreenEnglishTrans1 增量 #1 系統設計補建。
