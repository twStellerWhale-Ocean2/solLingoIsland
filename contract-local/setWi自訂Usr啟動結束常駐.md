# I. 主旨目的

定義使用者啟動與結束 [sysScreenTrans系統] 常駐狀態的配置作業。

# II. 參考準備

* 前置：[setWi自訂Usr安裝設定金鑰] 已完成。

# III. 內容程序

1. 雙擊 `ScreenTrans.exe`：程式啟動、不開主視窗、系統匣出現常駐圖示；重複啟動時提示已在執行（單一實例）。
2. 系統匣圖示右鍵選單：顯示狀態（金鑰是否備妥）、關於、結束。
3. 點選「結束」：釋放熱鍵註冊、程序完全退出、圖示消失。
4. （選配）隨開機啟動：使用者自行將捷徑加入「啟動」資料夾。

**可驗證結果**：啟動後系統匣圖示出現且熱鍵可用；結束後程序不存在、熱鍵釋放。
